# Cara Ajukan Permohonan

## Langkah-langkah

1. Buka menu **Pengajuan Informasi**
2. Pilih **Permohonan**
3. Isi form
4. Upload dokumen
5. Klik **Submit**

Sistem akan menampilkan nomor tiket.

---

## Catatan

- Data pribadi otomatis terisi
- Dokumen wajib sesuai ketentuan
