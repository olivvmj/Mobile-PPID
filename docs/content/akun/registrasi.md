# Daftar Akun

Untuk menggunakan aplikasi, pengguna harus melakukan registrasi akun.

## Langkah-langkah

1. Buka aplikasi Mobile PPID
2. Klik **Daftar Akun Baru**
3. Isi seluruh data yang diminta
4. Klik **Daftar Akun**

## Catatan

- Semua field wajib harus diisi
- Data harus sesuai identitas
